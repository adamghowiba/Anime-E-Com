## Features

- SvelteKit for routing
- Typescript for peace of mind
- SCSS
- Headless API intergration
- Lazy loading images
- Animations 
- Login
- Account mangement
- Cart & Checkout
- Stores
- Carousel (Slider)
- Toasts (Notifications)
- Loading Indicators
- Componentized
- Instant Search
- Filtering / Sorting
- Skeletons
- Responsive

## Tech Stack

- [Svelte](https://svelte.dev/)
- [SvelteKit](https://kit.svelte.dev/)
- [CommerceJS](https://commercejs.com/)
- SCSS
